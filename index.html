📌 Panduan Penggunaan Form Konfirmasi Project

Tahap 1: Pengisian Konfirmasi Project

Setiap PIC Wil/Cab wajib memeriksa dan melakukan konfirmasi data project kelolaannya masing-masing melalui halaman Input Konfirmasi Proyek.

Tahap 2: Manajemen Perubahan Data (Konfirmasi / Drop)

Sebelum database dikunci, Wilayah/Cabang melakukan konfirmasi apakah project tersebut Berjalan atau Batal, dan memiliki wewenang merubah rincian Usul entitas/ENP.

Tahap 3: Review Eksekutif Konsolidasi

Setelah seluruh status usulan proyek terkonfirmasi, perwakilan Wilayah diwajibkan melakukan *cross-check* menyeluruh pada halaman Summary Eksekutif.

Tahap 4: Pernyataan Validasi & Kunci Database

Apabila data cocok, sebagai bentuk komitmen, Wilayah (Setingkat Dept. Head) wajib melakukan validasi di halaman Lembar Validasi untuk mengunci akses edit.
