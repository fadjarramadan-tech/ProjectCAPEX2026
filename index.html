📌 Panduan Penggunaan Form Konfirmasi Project CAPEX Wil-Cab 2026

Tahap 1: Pengisian Konfirmasi Project

Setiap PIC Wil/Cab wajib memeriksa dan melakukan konfirmasi data project kelolaannya masing-masing melalui halaman Input Konfirmasi Proyek.

Tahap 2: Manajemen Perubahan Data (Konfirmasi / Drop)

Sebelum database dikunci, Wilayah/Cabang melakukan konfirmasi apakah project tersebut Berjalan atau Batal, dan Wilayah/Cabang memiliki kewenangan untuk mengubah rincian Usul Perubahan Entitas, Usul Perubahan ENP, Rencana realisasi atau menghapus log konfirmasi terhadap data yang telah diinput konfirmasi sebelumnya.

Tahap 3: Review Eksekutif Konsolidasi

Setelah seluruh status usulan proyek terkonfirmasi, perwakilan Wilayah diwajibkan melakukan *cross-check* menyeluruh pada halaman Summary Eksekutif.

Tahap 4: Pernyataan Validasi & Kunci Database

Apabila data project beserta rencana sebaran realisasinya telah cocok, sebagai bentuk komitmen, Wilayah (Setingkat Dept. Head) harus melakukan validasi dan penguncian di halaman Lembar Validasi. Maka hak akses edit wilayah otomatis akan ditutup by system.
